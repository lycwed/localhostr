# localhostR

This tool is made to display your localhost's projects in a pretty way in your browser.


## How to

1/ Download the zip.
2/ Unzip in the folder of your localhost.
3/ Don't change the folder name 'LOCALHOSTR'.
4/ Check you don't have any file call 'index.php' at the root
5/ Open it in your browser (ex: http://localhost:8080/LOCALHOSTR)
6/ That's it! Enjoy ^
